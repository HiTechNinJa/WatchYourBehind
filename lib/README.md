# 库目录 (Library Directory)

此目录用于放置项目特定的库文件。
This directory is used for project-specific library files.

PlatformIO会自动处理这里的库依赖。
PlatformIO will automatically handle library dependencies here.

每个库应该放在单独的子目录中，例如:
Each library should be in a separate subdirectory, for example:

```
lib/
  └── MyRadarLib/
      ├── MyRadarLib.h
      └── MyRadarLib.cpp
```

更多信息请访问: https://docs.platformio.org/page/projectconf/section_env_library.html
For more information, visit: https://docs.platformio.org/page/projectconf/section_env_library.html
